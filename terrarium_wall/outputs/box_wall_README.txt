Box wall tile set
=================

Panel surface algorithm: one full-face Voronoi field for each box side length, subdivided into separate printable panels.
Each panel has a small shallow back-side label such as S11 or L36; labels are not visible from the front.
Outer column panels include a black 45-degree corner filler lip; bottom-edge tube loops are clipped so the box can sit flat.
Target box: 14.75 in x 26.75 in x 12.00 in tall.
Base-height allowance used in panel math: 0.167 in per end.
Short face clear span: 14.42 in = 3 panels at 4.81 in wide.
Long face clear span: 26.42 in = 6 panels at 4.40 in wide.
Wall height: 3 rows at 4.00 in high.

Panel counts for one short face: 9 panels; one long face: 18 panels. Print two copies of each face set for a full box.
Corner posts: print 12 four-inch post sections for four 12-inch corners; each post has a top tab and bottom socket for stacking.
Straight seam connectors: about 156 two-stud connectors for panel-to-panel seams if every socket pair is connected.

Each panel folder contains shared beige/white/orange tube STLs plus two black base choices: solid and perforated.
For a solid panel, import the solid black base and the three tube STLs. For an airflow panel, import the perforated black base and the same three tube STLs.
PNG previews for the full short and long side layouts are in outputs/previews.
